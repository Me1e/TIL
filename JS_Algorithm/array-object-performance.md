# 배열과 오브젝트의 성능 평가

## Big O of Object

- Insertion - O(1)
- Removal - O(1)
- Searching - O(N)
  - value로 접근
- Access - O(1)
  - key로 접근

## Big O of Object Methods

- Object.keys - O(N)
- Object.values - O(N)
- object.entries - O(N)
- hasOwnProperty - O(1)

## Big O of Arrays

- Insertion
  - Push - O(1)
  - Shift - O(N)
- Removal
  - Pop - O(1)
  - Unshift - O(N)
- Searching - O(N)
- Access - O(1)
